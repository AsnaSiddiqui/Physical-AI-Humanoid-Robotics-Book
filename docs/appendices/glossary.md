# Glossary of Terms

This glossary provides definitions for key terms used throughout the book.

## A

**Action**: A ROS 2 communication pattern for long-running tasks with feedback and goal management.

## D

**Digital Twin**: A virtual replica of a physical robot used for simulation, testing, and optimization.

## G

**Gazebo**: A 3D simulation environment for robotics with accurate physics and rendering.

## I

**Isaac Sim**: NVIDIA's robotics simulation platform with photorealistic environments.

**Isaac ROS**: Hardware-accelerated perception and navigation packages for ROS.

## L

**LLM**: Large Language Model - a type of AI model trained on large text datasets.

## N

**Node**: A process that performs computation in ROS.

## R

**ROS 2**: The Robot Operating System version 2, a flexible framework for robot software development.

## S

**SDF**: Simulation Description Format - XML format used by Gazebo for simulation models.

## T

**Topic**: A ROS 2 communication mechanism based on publish/subscribe pattern.

## U

**URDF**: Unified Robot Description Format - XML format for representing robot models.

## V

**VLA**: Vision-Language-Action system - integrating visual perception, language understanding, and robotic action.
